# 🎧 Song Mood Classifier

This project predicts song mood using Machine Learning and allows users to search songs based on feelings using NLP.

## Features
- 🎧 Mood prediction using audio features
- 💭 Emotion-based input (love, sad, party)
- 🤖 Hybrid ML + NLP system
- 🌐 Streamlit UI

## Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit

## How to Run
streamlit run app.py
